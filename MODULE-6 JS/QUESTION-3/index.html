Question 3 : What is negative Infinity?

In JavaScript, negative infinity is a special value represented by the keyword `Number.NEGATIVE_INFINITY`.

It is a predefined constant that represents the lowest possible value, which is infinitely small.

ex:
var x = Number.NEGATIVE_INFINITY;

console.log(x); // Output: -Infinity
console.log(x < 0); // Output: true
console.log(x > -1000); // Output: false
console.log(x + 10); // Output: -Infinity
